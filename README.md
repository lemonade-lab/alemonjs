## Almeon 阿柠檬跨平台开发框架机器人

> 支持 QQ、QQ 频道、Discord、米游社大别野、KOOK

[☞Alemon 文档](https://alemonjs.com)

| 分支  | 类型   | 说明                 |
| ----- | ------ | -------------------- |
| main  | 机器人 | 机器人实例           |
| almeo | 核心   | 逻辑块               |
| redis | 数据库 | 特殊连接独立         |
| cli   | 脚手架 | 开发模板脚手架       |
| web   | 网站   | 共享平台插件贡献地址 |

## 快速开始

```shell
npm init almeon@latest -y #安装
```

```shell
cd alemon-bot #进入
```

```shell
npm run app qq #启动
```
