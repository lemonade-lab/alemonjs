## USE

- kook

```sh
yarn add @alemonjs/kook
```

- alemon.config.yaml

```sh
kook:
  # 令牌
  token: ''
  # 使用 user_key
  master_key:
     - 'xxx'
  # 使用 user_id
  master_id:
     - 'yyy'
```
