# [https://lemonade-lab.github.io/alemonjs.com/](https://lemonade-lab.github.io/alemonjs.com/)

跨平台开发的事件驱动机器人

## USE

- discord

```sh
yarn add @alemonjs/discord
```

- alemon.config.yaml

```sh
discord:
  # 令牌
  token: ''
```

> 完整的

```sh
discord:
  # 令牌
  token: ''
  # 前缀
  intent: []
  # 主人
  master_id: []
  # 活动
  shard: null
```
